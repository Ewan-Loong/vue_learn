# vue_learn

A universal backend management system built on libraries such as Vue3, Antd Vue4, VueRuter, etc

---

## 项目包管理
```
npm install
```

### 用于开发的编译和热重载
```
npm run serve
```

### 编译和缩小以用于生产
```
npm run build
```

### 整理和修复文件
```
npm run lint
```

## 目录含义
* public web图标等公共资源
* src 主目录
  * assets 全局css样式
  * components 表格、表单等基本组件
  * plugins 自定义插件
  * routes 路由管理
  * utils 工具js
  * views 模板页面
